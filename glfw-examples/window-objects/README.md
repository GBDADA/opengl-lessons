## Задание: "Менеджер окон с разными режимами"

Напиши программу, которая демонстрирует различные способы создания окон и управления ими.

### Основные требования:

1. **Создай главное окно** с возможностью выбора режима через аргументы командной строки:
   - `--mode=windowed` — обычное окно (по умолчанию)
   - `--mode=fullscreen` — настоящий полноэкранный режим
   - `--mode=borderless` — оконный полноэкранный (без рамок, с текущим разрешением монитора)

2. **Продемонстрируй использование разных window hints** в зависимости от режима:
   - Размер окна: 1280x720
   - Заголовок окна должен отображать текущий режим
   - Настрой декорирование, возможность изменения размера, плавающий режим (always-on-top) через отдельный флаг `--floating`
   - В полноэкранном режиме отключи авто-сворачивание при потере фокуса

3. **После создания окна выведи в консоль**:
   - Реальный размер окна (window size)
   - Размер framebuffer (физические пиксели)
   - Атрибуты окна: декорировано ли, изменяем ли размер, в полноэкранном ли режиме

4. **Реализуй интерактивное управление** (через клавиши):
   - `F` — переключение между оконным и полноэкранным режимом (с сохранением позиции и размера для оконного)
   - `B` — переключение в borderless fullscreen
   - `R` — восстановление из свернутого/максимизированного состояния
   - `M` — максимизация окна (только в оконном режиме)
   - `I` — свернуть окно
   - `Esc` — выход

5. **Покажи разницу между hard и soft constraints**:
   - Попробуй запросить OpenGL контекст с конкретными параметрами (например, версия 3.3 Core Profile)
   - Выведи, какой контекст реально получился (версия, профиль)

6. **Для оконного режима** установи минимальный размер 640x360 и соотношение сторон 16:9

  - Decorated: true
  - ...
