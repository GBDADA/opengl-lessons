## Задание: "Игровой прототип с виртуальным разрешением"

### Требования

#### 1. **Виртуальное разрешение**
- Задай константы `VIRTUAL_WIDTH = 1920` и `VIRTUAL_HEIGHT = 1080`
- Вся игровая логика и GUI работают в этих координатах
- При отрисовке происходит **масштабирование** на реальное окно с сохранением пропорций
- Чёрные полосы (letterbox) там, где пропорции не совпадают

#### 2. **Выбор разрешения через аргументы командной строки**
```bash
./game --width=1280 --height=720
./game --width=2560 --height=1440
./game --fullscreen
```
- Если разрешение не указано — используется виртуальное (1920x1080)
- Должна быть возможность переключения между оконным и полноэкранным режимом (клавиша `F`)

#### 3. **Адаптивный GUI (прямоугольники с заливкой)**
Нарисуй следующие элементы в виртуальных координатах (1920x1080):
- **Верхняя панель** — серая полоса высотой 60 пикселей, растянутая на всю ширину
- **Боковая панель слева** — тёмно-синяя, шириной 250 пикселей, на всю высоту (с учётом верхней панели)
- **Область контента** — зелёный прямоугольник, который занимает всё оставшееся пространство
- **Кнопка "Play"** — красный прямоугольник 200x60 в правом нижнем углу области контента (не боковой панели) - (мне не нравится эта идея дипсика, хочу чтобы кнопка была по центру контента)
- **Текст "Resolution: WxH"** — просто вывести в консоль или на экран (можно через GLFW window title или через рисование текста — но текстом пока можно пренебречь, просто прямоугольники)

#### 4. **Взаимодействие**
- При клике мышью по области красной кнопки в консоль выводится сообщение "Play clicked!"
- Координаты клика должны пересчитываться из оконных в виртуальные
- Должна работать проверка попадания в область кнопки

#### 5. **Переключение разрешения в реальном времени**
- Клавиша `R` — циклически переключает разрешение окна:
  - 1280x720
  - 1920x1080 (виртуальное)
  - 2560x1440
- При переключении масштабирование должно корректно обновляться

### Структура кода (рекомендуемая)

```cpp
struct VirtualRenderer {
    int virtualWidth, virtualHeight;  // 1920x1080
    int windowWidth, windowHeight;    // текущее окно
    int fbWidth, fbHeight;            // физические пиксели
    float scale;                      // масштаб отображения
    int offsetX, offsetY;             // смещение для letterbox
};

// Функции:
void updateViewport(VirtualRenderer* vr, GLFWwindow* window);
void worldToScreen(VirtualRenderer* vr, float& x, float& y);  // виртуальные -> оконные
void screenToWorld(VirtualRenderer* vr, double& x, double& y); // оконные -> виртуальные
void renderUI(VirtualRenderer* vr);  // рисует прямоугольники в виртуальных координатах
```

### Ожидаемый вывод

При запуске должно появиться окно с масштабированным интерфейсом:
- Прямоугольники должны сохранять свои пропорции
- При растягивании окна интерфейс масштабируется, но не "плывёт"
- Клик по красной кнопке работает
