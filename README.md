## Это учебный репозиторий

### Цель

Научиться делать игры, ну и просто весело тратить своё свободное время

### Сборка

Собранные программы будут находится в папке `build/bin/`
Для сборки использую скрипт fast-build.sh (или alias fb)
Пример - очистка папки build, сборка и запуск определённого проекта
`fb -c # Очистить папку build`
`fb -q -b -p "Название папки с проектом" -l # Если нету аргументов командной строки`
`fb -q -b -p "Название папки с проектом" -- аргументы # Если есть аргументы`
