## Уроки по OpenGL

В папке `tutorials-masandilov/` находятся мои конспекты и примеры кода
по урокам с сайта [masandilov.ru](https://masandilov.ru/opengl).

**Важно:** Код написан в учебных целях, повторяет логику уроков,
но может отличаться в деталях (списывать подчистую было бы скучно, так что буду стараться что-то от себя делать)

Оригинальные уроки принадлежат автору сайта.
